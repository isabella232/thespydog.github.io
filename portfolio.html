Open Source Projects

FNA

A reimplementation of the Microsoft XNA Game Studio 4.0 Refresh libraries with a focus on accuracy and portability. Primarily developed and maintained by Ethan "flibitijibibo" Lee.

Dozens of indie games use this framework, including Celeste, TowerFall, FEZ, Owlboy, Axiom Verge, Bastion, and Rogue Legacy.

I joined the project as a core developer in 2019. My contributions include implementing the touch input API, porting the library to new platforms (iOS, tvOS), developing a Metal graphics backend, improving documentation, and fixing various bugs. I also created and moderate the official FNA Discord server, the central hub for all FNA-related discussion.

(This is the framework I used to port Rogue Legacy to iOS! You can read more about that here.)


Ogmo Editor 3 CE

A tile-based level editor with a project-based workflow, originally developed by Matt Thorson and Noel Berry of EXOK Games.

Open-sourced and maintained by myself, Will Blanton, and Austin East.


TinyKVS

A tiny, dependency-free C# wrapper for iCloud Key Value Storage. Originally developed for use with Rogue Legacy: Wanderer Edition. Released under the Unlicense!
